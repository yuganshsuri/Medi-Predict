Our Mission

ML That Makes Medicine More Accessible

MediPredict is a machine learning research project built to demonstrate how data science can make early disease detection available to everyone — not just clinics.

What We Built

Bridging Clinical Data with Everyday Understanding

Every model in MediPredict was built from public clinical datasets, trained using scikit-learn, and evaluated against real patient outcomes. We care about accuracy — but also about making results interpretable.

This isn't a replacement for a doctor. It's a tool to help you ask better questions, understand your numbers, and take preventive action before symptoms escalate.

The entire stack is open, transparent, and built as a CS final-year project demonstrating the real-world potential of applied machine learning in healthcare.

Development Process

From Dataset to Diagnosis

Each model followed the same rigorous pipeline — from raw data to a production-ready classifier.

🗃️
Step 1 — Data Collection

Clinical Dataset Sourcing

All datasets sourced from UCI Machine Learning Repository and Kaggle clinical datasets — curated, peer-reviewed patient records with labeled outcomes.

🧹
Step 2 — Preprocessing

Data Cleaning & Feature Engineering

Handled missing values, normalized numerical features, encoded categorical variables, and applied best Ml algorithm

🔬
Step 3 — Model Selection

Algorithm Benchmarking

Compared Random Forest, SVM, XGBoost, Logistic Regression, KNN, and Neural Network classifiers per disease using cross-validation F1 scores.

⚙️
Step 4 — Tuning

Hyperparameter Optimization

Grid search and Bayesian optimization applied for each final model. Recall prioritized over precision to minimize false negatives in disease detection.

🚀
Step 5 — Deployment

Frontend Integration

Models pickled and served via a lightweight Python backend. The frontend provides a clean, accessible interface for non-technical users.

Technology

The Stack Behind MediPredict

Chosen for transparency, reproducibility, and open-source availability.

ML Core

scikit-learn

Primary ML library for all classifiers — Random Forest, SVM, Logistic Regression, and KNN models are built on sklearn's estimator API.

Boosting

XGBoost

Used for Stroke Risk and complex multi-feature models where ensemble gradient boosting outperformed standalone classifiers.

Data

Pandas & NumPy

All data wrangling, feature engineering, and statistical preprocessing was handled with pandas DataFrames and numpy array operations.

Backend

Python + Flask

Lightweight REST API serving serialized model predictions. Models pickled using joblib for fast cold-load serving.

Principles

Sustainable Tech

Open-source models, no cloud lock-in, no unnecessary compute overhead. MediPredict is designed to run efficiently on minimal hardware.
